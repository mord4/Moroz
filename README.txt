Привет, это проект Яндекс Лицея по PyQT5.

Проект: Создатель уведомлений

Есть несколько файлов:
  1) Файл с кодом Qt (код написан вручную), заносящий данные в БД + вспомогательный конструктор объектов
  2) Файлы для показа информации о созданных уведомлениях и показа документации(инструкции)
  3) Файлы с кодом проверки времени, тг ботом и создателем push-уведомлений в Windows 10
  4) Файл для получения информации из БД
  5) БД с двумя таблицами(в ней будут хранится созданные уведомления-напоминания)

Программа для создания напоминаний в Telegram или Windiws 10.
Инструкция по пользованию приложена в фале doc.txt, чтобы открыть его нажмите Alt + D, перед этим запустив программу.
